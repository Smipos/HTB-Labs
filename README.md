# HTB Labs

## Разбор трека Detecting Active Directory Attacks

Репозиторий содержит разборы лабораторных сценариев Hack The Box, связанных с Active Directory.

Цель проекта — продемонстрировать практические навыки анализа событий Windows.

| 01   | [Campfire-1](ActiveDirectory%20Track/01-campfire1/README.md) | В этом задании нужно изучить артефакты и логи с контроллера домена, а также артефакты с endpoint-устройства, с которого исходила активность Kerberoasting-атаки. Также предстоит разобраться, на что именно нужно обращать внимание для выявления Kerberoasting-атаки.|
